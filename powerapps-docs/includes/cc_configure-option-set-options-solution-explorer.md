Для определения параметра в области **Параметры** щелкните ![Добавить значок параметра](../maker/common-data-service/media/add-option-set-option-button.png), чтобы добавить параметр.

Для каждого параметра можно добавить следующие значения свойств

|Свойство|Описание|
|--|--|
|**Метка**|Локализуемый текст для описания параметра.|
|**Значение**|Числовое значение параметра. Перед ним указывается префикс, определенный в издателе решения для решения, с которым вы работаете. Этот префикс гарантирует, что данное значение будет отличаться от значений параметров, которые могут быть определены в другом решении и импортированы в вашу среду.|
|**Внешнее значение**|Это значение используется для виртуальных сущностей для сопоставления значения во внешнем источнике данных с этим параметром.|
|**Description;**|Место для записи назначения этого параметра. Это значение не отображается пользователям приложения.|
|**Цвет**|Задает цвет, используемый на диаграммах для этого параметра.<br />**Совет**. Используйте сайт, подобный [w3schools.com HTML Color Picker](https://www.w3schools.com/colors/colors_picker.asp), для определения шестнадцатеричного кода цвета.|

Используйте другие значки на панели инструментов параметров для выполнения следующих операций:

|Значок|Операция|
|--|--|
|![Значок удаления выбранного параметра](../maker/common-data-service/media/remove-option-solution-explorer.gif)|Удалить выбранный параметр|
|![Значок перемещения вверх выбранного параметра](../maker/common-data-service/media/move-selected-option-up-solution-explorer.png)|Переместить вверх выбранный параметр|
|![Значок перемещения вниз выбранного параметра](../maker/common-data-service/media/move-selected-option-down-solution-explorer.png)|Переместить вниз выбранный параметр|
|![Значок сортировки параметров по возрастанию](../maker/common-data-service/media/sort-option-set-option-asc-solution-explorer.png)|Сортировать параметры по метке в порядке возрастания|
|![Значок сортировки параметров по убыванию](../maker/common-data-service/media/sort-option-set-option-des-solution-explorer.png)|Сортировать параметры по метке в порядке убывания|
