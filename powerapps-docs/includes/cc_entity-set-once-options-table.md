|Параметр   |Описание  |
|---------|---------|
|**Действия**|Свяжите действия с записями для этой сущности.|
|**Последовательности операций бизнес-процесса**|Создайте последовательности операций бизнес-процесса для этой сущности. Дополнительные сведения: [Создание последовательности операций бизнес-процесса для стандартизации процессов](/flow/create-business-process-flow)|
|**Подключения**|Используйте функцию подключений, чтобы показать, как записи для этой сущности подключаются к записям других сущностей, для которых также включены подключения.|
|**Отзывы**|Предоставьте клиентам возможность оставлять отзывы о любой записи сущности или оценивать записи сущностей в пределах заданного диапазона оценок. Дополнительные сведения: [Включение поддержки отзывов или оценок для сущности](../maker/common-data-service/configure-entity-feedback.md)|
|**Примечания**|Добавьте примечания к записям для этой сущности. Примечания предоставляют возможность добавления вложений.|
|**Очереди**|Используйте сущность с очередями. С помощью очередей можно усовершенствовать маршрутизацию и распределение работ, предоставляя централизованный доступ к записям для этой сущности всем, кто обладает соответствующими полномочиями.|
|**Отправка сообщения электронной почты**|Отправляйте сообщения эл. почты, используя адрес, сохраненный в одном из полей для этой сущности. Если поле **Строка текста** с форматом **Электронная почта** еще не зарегистрировано для этой сущности, то новое будет создано при включении отправки сообщений электронной почты.|
